Unbounded is an animation first combat game prototype built around expressive motion and system-driven storytelling. The demo focuses on stylized martial arts combat inspired by Muay Boran, blending hand crafted animation with experimental gameplay mechanics that react to player behavior.  Unbounded explores how animation, timing, and game systems can communicate narrative indirectly, without relying on exposition or traditional cutscenes. This project represents my interest in animation, game development, and engineering through interactive design.
